Resumen de Temas

React

React es una biblioteca de JavaScript para construir interfaces de usuario. Fue desarrollada por Facebook y está disponible como un código abierto bajo la licencia MIT. React es utilizado por muchas compañías como Airbnb, Netflix, Instagram, Dropbox, Uber y muchas otras. React se basa en el concepto de componentes. Los componentes son bloques de construcción reutilizables que permiten a los desarrolladores crear interfaces de usuario complejas. Cada componente de React es una pieza aislada de código que tiene su propio estado, propiedades y métodos.

NodeJs

Node.js es una plataforma de tiempo de ejecución de JavaScript que se utiliza para construir aplicaciones web y de servidor. Fue desarrollado por Ryan Dahl en 2009 y está construido sobre el motor V8 de Google, que también se utiliza en el navegador Google Chrome. Node.js es gratuito y de código abierto, y está disponible para su descarga en el sitio web oficial.

API REST

Las API REST son un estilo arquitectónico para el diseño de servicios web que utilizan el protocolo HTTP para comunicarse. REST significa Transferencia de Estado Representacional (Representational State Transfer, en inglés) y se utiliza para describir cómo los recursos son direccionados y manipulados a través de una interfaz HTTP uniforme.

typeScript

TypeScript es un lenguaje de programación de código abierto desarrollado y mantenido por Microsoft. Es un superset de JavaScript, lo que significa que se basa en JavaScript pero agrega nuevas características y funcionalidades para ayudar a los desarrolladores a escribir código más seguro y escalable.
