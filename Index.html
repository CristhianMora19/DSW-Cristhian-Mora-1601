Todo Sobre Box Model

El Box Model es un concepto utilizado en el diseño de sitios web para evidenciar y describir como se estructuran y se dimensiona los elementos de una pagina HTML

Normalmente esta compuesta por 4 partes, Content, El Padding Border, y margin, estas capas se super ponen unas encima de las otras con el fin de formar el respectivo elemento HTML.


Content: es la parte inferior de los elementos HTML, en el cual se contiene la informacion pricipal tales como Texto, Imagenes, Videos y Demas.

Padiding: se habla de un espacio trasparente el cual se utiliza para agregar espacios en blanco entre contenido y borde, como es de esperarse se encuentra situado o ubicado al rededor del contenido

Border: conocido como linea o marco que se encuentra rodenado el elemento y es utiizado para agregar estilos o efectos visuales al rededor del contenido

Margin: normalmente es utilizada para agregar espacions entre los elementos adyacentes se encuentra fuera del borde siendo un espacio trasparente.

Cada una de las anteriores explicaciones se puede evidencia en la imagen en la cualse encuentra todo explicado de manera grafica y sencilla con el fin de que se comprenda el tema de manera didactica.

Ademas de esto es importante el Box Model dentro de Diseño Web ya que permite a el programador controlar y desarrollar con precision los elementos de la pagina.
